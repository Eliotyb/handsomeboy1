# 🌐 Agent Browser - 浏览器自动化

## 技能概述

提供完整的浏览器自动化能力，支持自动浏览、表单填写、数据抓取、页面交互、截图验证等操作。

## 核心能力

### 1. 自动浏览与导航
- 打开指定 URL 并等待页面加载
- 前进/后退/刷新页面
- 多标签页管理（打开/切换/关闭）
- 页面滚动（指定方向/距离/元素定位）

### 2. 表单自动填写
- 识别并填充文本输入框
- 选择下拉菜单选项
- 勾选/取消复选框和单选按钮
- 文件上传操作
- 表单提交（点击按钮或回车）

### 3. 数据抓取
- 提取页面文本内容
- 获取元素属性值
- 抓取表格数据
- 提取链接列表
- 截图保存（全页/区域/元素）

### 4. 页面交互
- 点击按钮/链接/元素
- 悬停触发事件
- 键盘输入模拟
- 拖拽操作
- 等待元素出现/消失

## 使用场景

| 场景 | 说明 |
|------|------|
| **数据采集** | 批量抓取商品信息、文章内容、表格数据 |
| **自动填表** | 自动化注册、报名、订单提交等重复性表单操作 |
| **UI 测试** | 验证页面功能、检查元素状态、回归测试 |
| **监控巡检** | 定期访问页面检查状态变化 |
| **内容提取** | 从复杂页面结构中提取目标信息 |

## 工作流程

```
1. 接收任务指令 → 解析目标URL和操作需求
2. 启动浏览器 → 导航到目标页面
3. 等待加载完成 → 快照分析页面结构
4. 执行操作序列 → 点击/输入/滚动/等待
5. 验证结果 → 截图/提取数据/确认状态
6. 输出报告 → 返回操作结果和数据
```

## 使用示例

### 示例1：自动搜索并抓取结果
```
用户: 帮我打开百度搜索"Vue3教程"，抓取前5条结果的标题和链接
执行:
1. browser_navigate → https://www.baidu.com
2. browser_snapshot → 定位搜索框
3. browser_type → 输入"Vue3教程"
4. browser_click → 点击搜索按钮
5. browser_wait_for → 等待结果加载
6. browser_snapshot → 提取搜索结果列表
7. 返回: 标题+链接的结构化数据
```

### 示例2：自动登录流程
```
用户: 帮我自动登录这个网站 xxx.com
执行:
1. browser_navigate → 目标登录页
2. browser_snapshot → 识别用户名/密码输入框
3. browser_type → 填写凭据
4. browser_click → 点击登录按钮
5. browser_wait_for → 等待跳转完成
6. browser_take_screenshot → 截图验证登录状态
7. 返回: 登录成功/失败状态 + 当前页面信息
```

### 示例3：批量表单填写
```
用户: 把这些Excel数据批量填到这个在线表单里
执行:
1. 读取Excel数据源
2. 循环每一行数据:
   - browser_navigate → 表单页面
   - browser_type → 逐字段填写
   - browser_click → 提交表单
   - browser_wait_for → 确认提交成功
   - 记录结果
3. 输出: 成功/失败汇总报告
```

## 注意事项

- ⚠️ 操作前先 `browser_snapshot` 分析页面结构，不要盲目操作
- ⚠️ 使用 `browser_wait_for` 等待动态内容加载完成
- ⚠️ 复杂页面建议分步执行，每步验证后再继续
- ⚠️ 遇到弹窗/验证码等反爬机制时及时告知用户
- ⚠️ 敏感操作（删除/支付）前务必二次确认
