# 📄 MiniMax Office - 强化版Office三件套

## 技能概述

提供 Word / Excel / PDF 三大办公格式的专业处理能力，支持创建、编辑、转换、批量操作等全流程自动化。

## 一、Word 文档处理 (docx)

### 1.1 文档类型模板

| 类型 | 适用场景 | 结构特点 |
|------|----------|----------|
| **正式公文** | 通知/报告/请示 | 红头+正文+落款+附件 |
| **学术论文** | 毕业论文/期刊投稿 | 摘要/目录/章节/参考文献 |
| **商业计划书** | BP/融资材料 | 执行摘要/市场/产品/财务/团队 |
| **会议纪要** | 项目会议/周报 | 基本信息/议题/决议/待办 |
| **合同协议** | 合作/劳务/保密 | 条款定义/权利义务/签署区 |
| **简历/CV** | 求职/申请 | 个人信息/经历/技能/自我评价 |
| **操作手册** | 产品文档/SOP | 目录/步骤/截图/FAQ |

### 1.2 文档结构规范

#### 正式公文格式
```markdown
# XXXX公司文件

**签发人**: XXX                 **日期**: 2025年XX月XX日

---

## 关于XXXX的通知

各部门/各同事：

正文内容...

特此通知。

<br>

<div align="right">
XXX部门<br>
2025年XX月XX日
</div>

---
**附件**: 1.XXX清单  2.XXX表格
**抄送**: XXX/XXX
```

#### 学术论文格式
```markdown
# 论文标题（中文）
# English Title

**作者**: XXX  **指导教师**: XXX  **学院**: XXX

---

## 摘要
[300字中文摘要]

## Abstract
[English abstract, 250 words]

## 1 引言 (Introduction)
```

## 二、Excel 数据处理 (xlsx)

### 2.1 表格类型模板

| 类型 | 场景 | 特征列 |
|------|------|--------|
| **人员名单** | 名册/通讯录 | 姓名/性别/电话/邮箱/部门 |

### 2.2 公式与函数库

#### 常用函数速查
| 类别 | 函数 | 用途 | 示例 |
|------|------|------|------|
| **查找** | VLOOKUP/XLOOKUP | 匹配查找 | `=VLOOKUP(A2,表,2,FALSE)` |
| **条件** | IF/IFS/SWITCH | 条件判断 | `=IF(A2>60,"及格","不及格")` |
| **统计** | SUMIF/COUNTIF/AVERAGEIF | 条件统计 | `=SUMIF(B:B,"销售",C:C)` |
| **文本** | LEFT/RIGHT/MID/CONCAT | 文本截取拼接 | `=CONCAT(A2,"-",B2)` |
| **日期** | DATEDIF/EDATE/WEEKDAY | 日期计算 | `=DATEDIF(A2,TODAY(),"Y")` |
| **数组** | UNIQUE/FILTER/SORT | 动态数组 | `=UNIQUE(A2:A100)` |
| **逻辑** | AND/OR/NOT/XOR | 组合逻辑 | `=AND(A2>0,B2<100)` |

## 使用示例

### 示例1：批量生成Word合同
```
用户: 我有100个人的信息，需要生成100份劳动合同
→ 流程:
  1. 读取 Excel 人员名单
  2. 加载 Word 合同模板({姓名}/{岗位}/{薪资}占位符)
  3. 循环替换变量 → 生成100份 docx
  4. 输出到 output/contracts/ 目录
```

### 示例2：Excel 数据分析
```
用户: 帮我分析这份销售数据
→ 流程:
  1. 读取 xlsx 文件
  2. 数据清洗: 去重/补缺/格式统一
  3. 分析维度: 月度趋势/产品TOP10/区域分布
  4. 生成: 数据透视表 + 可视化图表
  5. 输出: 分析报告(PDF格式)
```

### 示例3：PDF 合并与水印
```
用户: 把这20个PDF合成一个，加上水印
→ 流程:
  1. 按 file_01 ~ file_20 顺序合并
  2. 添加半透明文字水印 "内部资料-机密"
  3. 设置打开密码
  4. 压缩优化(减小体积)
  5. 输出: merged_final.pdf
```
